# QBS Звонки — Frontend

Next.js frontend для сервиса аналитики телефонных звонков QBS Звонки.

## Setup

```bash
npm install
npm run dev
```

The application will run on http://localhost:3000

## Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```
