## Реализованные требования

### ✅ Requirement 10.1: Линейный график времени разговоров
- Использован `LineChart` из Recharts
- Отображает фактическое время разговоров по дням
- Ось X: даты в формате DD.MM
- Ось Y: время в минутах

### ✅ Requirement 10.2: Линия целевых значений
- Пунктирная серая линия (`strokeDasharray="5 5"`)
- Использует `targetTalkTimeMinutes` из данных
- Отображается без точек для чистоты визуализации

### ✅ Requirement 10.3: Цветовое кодирование точек
- Зеленый: выполнение >= 100% (`hsl(142, 76%, 36%)`)
- Желтый: выполнение >= 80% и < 100% (`hsl(48, 96%, 53%)`)
- Красный: выполнение < 80% (`hsl(0, 84%, 60%)`)
- Реализовано через `CustomDot` компонент
- Использует утилиту `getColorByPercentage` из `@/lib/kpi-utils`

### ✅ Requirement 10.4: Tooltip с детальной информацией
- Кастомный `CustomTooltip` компонент
- Показывает:
  - Дату в формате "01 января 2024"
  - Фактическое время (мин)
  - Целевое время (мин)
  - Процент выполнения с цветовым кодированием
- Стилизован с использованием shadcn/ui Card

### ✅ Responsive дизайн
- Использует `ResponsiveContainer` из Recharts
- Адаптируется под ширину родительского контейнера
- Фиксированная высота 400px

### ✅ Loading состояние
- Показывает `ChartSkeleton` при `loading={true}`
- Skeleton имеет ту же высоту, что и график (400px)

### ✅ Code splitting
- Компонент помечен как `"use client"`
- Может быть загружен через `React.lazy()` при необходимости
- Recharts загружается только при использовании компонента

## Технические детали

### Зависимости
- `recharts@3.8.1` - установлена в apps/app/package.json
- `@calls/ui` - для Card и Skeleton
- `@/lib/kpi-utils` - для getColorByPercentage

### Компоненты Recharts
- `LineChart` - основной контейнер графика
- `Line` - линии для фактического и целевого времени
- `XAxis`, `YAxis` - оси координат
- `CartesianGrid` - сетка
- `Tooltip` - всплывающая подсказка
- `Legend` - легенда графика
- `ResponsiveContainer` - адаптивный контейнер

### Accessibility
- Все элементы имеют `aria-label` атрибуты
- График доступен для screen readers
- Цветовое кодирование дополнено текстовой информацией

## Интеграция

Компонент экспортируется из `apps/app/src/components/features/kpi/index.ts`:

```typescript
export { TrendChart } from "./trend-chart";
```

Использование:

```typescript
import { TrendChart } from "@/components/features/kpi";

<TrendChart data={dailyKpiData} loading={isLoading} />
```

## Состояния компонента

1. **Нормальное**: Отображает график с данными
2. **Loading**: Показывает skeleton
3. **Пустые данные**: Возвращает `null` (не рендерится)
