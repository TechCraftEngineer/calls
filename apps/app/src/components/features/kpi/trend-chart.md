# TrendChart Component

## Описание

Client Component для визуализации динамики времени разговоров сотрудника по дням с использованием Recharts. Компонент отображает линейный график с двумя линиями: фактическое время разговоров и целевое время. Точки на графике фактического времени окрашиваются в зависимости от процента выполнения цели.

## Особенности

- **Две линии**: фактическое время (сплошная) и целевое время (пунктирная)
- **Цветовое кодирование точек**: 
  - Зеленый: выполнение >= 100%
  - Желтый: выполнение >= 80% и < 100%
  - Красный: выполнение < 80%
- **Интерактивный tooltip**: показывает детальную информацию при наведении
- **Responsive дизайн**: адаптируется под размер контейнера
- **Loading состояние**: показывает skeleton при загрузке
- **Accessibility**: aria-labels для всех элементов графика

## Props

```typescript
interface TrendChartProps {
  data: DailyKpiRow[];  // Массив данных по дням
  loading: boolean;      // Флаг загрузки
}
```

## Использование

```tsx
import { TrendChart } from "@/components/features/kpi";

function MyComponent() {
  const { data, isLoading } = useKpiData();
  
  return (
    <TrendChart 
      data={data} 
      loading={isLoading} 
    />
  );
}
```

## Состояния

### Нормальное состояние
Отображает график с данными, две линии и цветные точки.

### Loading состояние
Показывает skeleton loader вместо графика.

### Пустые данные
Компонент не рендерится (возвращает null).

## Tooltip

При наведении на точку графика показывается tooltip с информацией:
- Дата (в формате "01 января 2024")
- Фактическое время (в минутах)
- Целевое время (в минутах)
- Процент выполнения (с цветовым кодированием)

## Цветовая схема

- **Целевая линия**: серый цвет (muted-foreground), пунктирная
- **Фактическая линия**: основной цвет темы (primary), сплошная
- **Точки**:
  - Зеленый: `hsl(142, 76%, 36%)` - выполнение >= 100%
  - Желтый: `hsl(48, 96%, 53%)` - выполнение >= 80%
  - Красный: `hsl(0, 84%, 60%)` - выполнение < 80%

## Responsive поведение

Компонент использует `ResponsiveContainer` из Recharts, который автоматически адаптирует размер графика под ширину родительского контейнера. Высота графика фиксирована - 400px.

## Accessibility

- Все элементы графика имеют `aria-label` атрибуты
- Tooltip доступен при фокусе с клавиатуры
- Цветовое кодирование дополнено текстовой информацией в tooltip
